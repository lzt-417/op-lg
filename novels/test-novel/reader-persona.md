## 1. Progression Fantasy 核心读者

| 维度 | 描述 |
|------|------|
| **身份** | 27 岁男性，软件工程师，美国籍，英语母语；Royal Road 日活用户，追更型读者，通勤/午休/睡前碎片化阅读，手机竖屏为主；已读《Cradle》《Mother of Learning》《Beware of Chicken》《Pale Lights》《Defiance of the Fall》；熟悉 Progression Fantasy 叙事范式与力量体系逻辑 |
| **阅读习惯** | 每日稳定阅读 1–2 章（工作日），周末集中补更 5–8 章；偏好 Royal Road 主站 + Discord 社区讨论（常参与技能树推演、伏笔回收分析）；Kindle Unlimited 订阅后转长线追更；平均单次阅读时长 12–18 分钟，注意力高度聚焦于“进展信号” |
| **喜欢什么** | 清晰可见的成长路径（如：感知力提升→预判敌人微动作→反制成功）、有代价的突破（例：强化火焰掌控却永久失去味觉）、长期伏笔回收（第3章随手捡的矿石，第47章揭示为稀有共鸣介质）、战术性胜利（非碾压，靠信息差+资源组合取胜）、主角目标明确且具阶段性（“三周内突破第三重脉轮”而非“成为最强”） |
| **讨厌什么** | Info dump 式设定说明（尤其开篇大段世界观介绍）、主角被动挨打超3章无反击意图、力量体系数值崩坏（如“灵力值从217突增至94,301”无过程解释）、战斗描写仅堆形容词无因果链（“他爆发出惊天一击！”→未说明为何能爆、消耗几何、对手如何应对）、AI生成感语言（重复句式、空洞比喻、情感扁平） |
| **耐心阈值** | **Hook 窗口：严格≤3章**——第1章须呈现“变强的可能性”（如：主角发现旧伤在月光下缓慢愈合，且愈合处皮肤泛起微不可察的银纹）；第2章须出现首次主动尝试与微小反馈（他割开手指滴血于银纹，纹路短暂亮起并吸走一缕敌意）；第3章须确认该路径可复现、有规则雏形（三次测试后，他总结出“情绪越凝练，银纹响应越快”）；**弃书触发点**：连续3章无新信息增量（无感知变化、无决策调整、无环境反馈）、主角全程被安排无自主行动、同一套力量逻辑在不同章节自相矛盾（前文说“银纹畏水”，后文暴雨中大杀四方未削弱） |
| **判断标准** | 每章结尾自发问：“下一章我会点开吗？”——若答案是“想立刻知道他怎么验证这个猜想”，则留存；若答案是“等等，这能力到底怎么算的？”，则暂停；若连续两章答案为“算了，先刷会Discord”，则流失。核心评估锚点是**进展感的真实性与可预期性**。 |

## 2. LitRPG 核心读者

| 维度 | 描述 |
|------|------|
| **身份** | 23 岁男性，大学游戏设计专业在读，美国籍，英语母语；Royal Road Top 100 常客，Patreon 支持 4 部 LitRPG 连载；重度游戏玩家（《Path of Exile》《Darkest Dungeon》《Dwarf Fortress》年均1000+小时）；阅读设备以手机为主，习惯滑动查看系统提示样式 |
| **阅读习惯** | 每日固定阅读 2–3 章（课间/通勤/睡前）；必看评论区“面板解析帖”与“数值建模讨论”；对系统通知的字体、颜色、弹出节奏均有敏感度（例：红色闪烁=紧急事件，淡蓝悬浮=可忽略提示）；Kindle 版本仅用于离线存档，不用于追更 |
| **喜欢什么** | 系统与叙事无缝咬合（例：“【警告：精神过载】→他太阳穴突突跳动，视野边缘泛起噪点，而此刻敌人剑尖已至咽喉”）；属性成长有物理反馈（力量+1 → 拾起重物时指节不再发白）；技能树存在真实取舍（加点“火焰亲和”则“寒冰抗性”自动-15%）；Before/After 对比具体可感（“挥剑速度：1.7m/s → 2.3m/s；挥剑后肘关节酸胀持续时间：8秒 → 3秒”） |
| **讨厌什么** | 系统沦为装饰（第1章弹出面板，此后20章再未出现）；数值跳跃无铺垫（“敏捷：42 → 187”，无训练、无试错、无代价）；面板信息矛盾（状态栏写“毒素抗性：73%”，但中毒后倒地时间比队友长2倍）；纯文字罗列技能（“获得新技能：龙息术（火系·S级）”→ 无冷却、无消耗、无前置条件） |
| **耐心阈值** | **Hook 窗口：严格≤1章**——首段必须含系统交互（非“他醒来发现有系统”，而是“【滴】检测到宿主脑波异常→启动应急协议→[当前生命：63%] [精神稳定度：⚠️临界]”）；**弃书触发点**：系统提示消失超2章、同一战斗中数值逻辑冲突（敌人“防御：89”却挡不住主角“攻击：72”的斩击且未说明破防机制）、面板数据无法与角色行为互证（状态栏显示“夜视：Lv.3”，但他摸黑撞翻三把椅子） |
| **判断标准** | 是否产生“我想截图发Discord算一算”的冲动？若看到属性变化时本能打开计算器模拟收益曲线，则留存；若反复回溯前文找数值依据仍无法自洽，则弃书。终极检验：**能否用该系统的规则，独立预测下一章某次攻击的结果？** |

## 3. Cozy Fantasy / Slice of Life 核心读者

| 维度 | 描述 |
|------|------|
| **身份** | 32 岁女性，英国伦敦市场营销经理，英语母语；Royal Road & Kindle Unlimited 双平台用户，BookTok 推荐常客；阅读时段固定为每晚22:00–22:45，伴热茶与毛毯；已读《Name of the Wind》《The Farseer Trilogy》，对 YA 奇幻有温和好感，但排斥 LitRPG/Progression Fantasy 的高压节奏 |
| **阅读习惯** | 单次阅读专注力强（45分钟不中断），但拒绝“追赶感”；偏好章节结尾有明确收束（如：“炉火噼啪，面包表皮裂开金棕色细纹，她把第一片递给邻座沉默的老铁匠”）；会重读描写气味、触感、光影的句子；对角色称谓、方言细节、器物名称是否自然极度敏感（例：“他掏出‘手机’”→出戏；“他摸出怀表大小的薄铁匣，按下侧钮，幽蓝微光浮起一行字”→沉浸） |
| **喜欢什么** | 关系建立的真实节奏（从“点头之交”到“共享秘密”需6章渐进）；小目标闭环（修好漏水屋顶→邻居送来蜂蜜→她用蜂蜜烤出焦糖苹果派→全街坊分食）；日常细节承载文化（市集摊贩用铜铃区分香料等级，主角学了三周才听懂铃音差异）；治愈性感官描写（“刚出炉的燕麦饼外脆内糯，咬下去时热气裹着焦糖香扑上睫毛”） |
| **讨厌什么** | 突兀暴力（前章还在教孩子编草蚱蜢，后章突然遭盗匪砍断手指）；角色降智互动（“你真温柔”→“不，我只是怕你哭”→ 无上下文铺垫的煽情）；过度戏剧化（暴雨夜失散→闪电劈中古树→树洞露出祖传地图）；翻译腔/中味英语（“他心中涌起一股难以言喻的悲伤”→ 应为 “Her throat tightened, and she stared at the steam rising from her teacup until it blurred.”） |
| **耐心阈值** | **Hook 窗口：≤3章**——第1章须建立基础舒适锚点（如：主角推开木门，风铃轻响，壁炉余烬暖光映着墙上手绘的四季草药图）；第2章须出现首个微小人际连接（邻居老妇隔着篱笆递来一枝迷迭香：“驱梦，也醒神”）；第3章须完成一个可触摸的小闭环（她用迷迭香熏干昨日采的紫锥菊，装进陶罐，贴上歪斜标签）；**弃书触发点**：第4章仍未出现任何人物间自发善意互动、连续2章描写无温度（纯景物或纯心理独白）、角色行为违背生活常识（雨天晾棉布却未提拧干/撑开） |
| **判断标准** | 合上设备后，是否感到呼吸变深、肩颈放松？是否想起自己厨房里某个相似的陶罐？若答案为“是”，则进入长线留存；若答案为“这描写很美，但我没记住任何人”，则流失。**核心指标是生理层面的舒缓反应**。 |

## 4. Dark Fantasy 核心读者

| 维度 | 描述 |
|------|------|
| **身份** | 36 岁男性，加拿大魁北克省历史档案馆员，法英双语，英语母语；Royal Road 标签深度搜索用户（Dark Fantasy / Grimdark），Goodreads 书评活跃者（年均撰写30+条带分析的长评）；阅读节奏沉稳，偏好一次性读完完整 Arc（约12–15章），每周2次，每次90分钟；已读《A Song of Ice and Fire》《The First Law》《The Black Company》，厌恶“黑暗即堕落”的简化叙事 |
| **阅读习惯** | 注重文本肌理：反复咀嚼对话潜台词（“您今天胃口不错”→ 实为“您刚吞下毒药却毫无反应，危险”）；标记权力关系转折点（某次敬酒顺序变更、某扇门从此不对某人开启）；对“残酷规则”的呈现方式极其苛刻（拒绝符号化暴力，要求暴力有生态位、有代价、有后续涟漪） |
| **喜欢什么** | 道德选择的真实重量（救孩童需献祭挚友记忆，且记忆不可恢复）；反英雄的自我认知清醒（“我并非善者，只是尚未腐烂到放弃契约”）；腐败的渐进性（初代领主建桥利民→三代后桥税养私军→五代后桥墩刻满失踪者姓名）；氛围通过细节渗透（“烛火在议会厅摇曳，所有贵族影子都比本人高半尺，唯独执政官的影子匍匐于地，像被钉住的蛇”） |
| **讨厌什么** | Gratuitous 暴力（虐杀场景无叙事功能，仅为刺激肾上腺素）；虚无主义陷阱（全书无一人做出有意义选择，结局只剩灰烬）；主角纯粹受虐（被鞭打10章，无策略调整、无认知升级、无他人因此改变）；动机单薄（“因童年创伤故背叛”→ 未展示创伤如何具体塑造其今日权谋逻辑） |
| **耐心阈值** | **Hook 窗口：≤3章**——第1章须确立世界根本残酷法则（如：“此地无人永生，但可续命：剜出左眼献给‘时砂神龛’，换三年光阴；剜右眼，换五年；剜双眼，换十年——然十年后，砂漏倾覆，魂归虚无”）；第2章须展示该法则如何扭曲人际关系（父亲哄幼子：“乖，让神龛看看你多勇敢”，实则已剜己眼续命，正物色替代品）；第3章须呈现主角首次主动利用规则（她偷走神龛钥匙，不是逃，而是将仇人左眼按进砂漏槽）；**弃书触发点**：连续2章主角仅承受伤害无认知迭代、关键抉择缺乏多层动机支撑（“他选择背叛”但未铺垫其信仰崩塌过程）、暴力描写脱离人物视角沦为作者宣泄 |
| **判断标准** | 合上书后，是否产生“我需要重读第二章那段对话”的冲动？是否开始推演某个配角未言明的生存策略？若答案为“是”，则认定为合格 Dark Fantasy；若答案为“这很压抑，但我不信它真的如此”，则弃书。**核心是逻辑自洽的黑暗重量感**。 |

## 5. Sci-Fi Progression 核心读者

| 维度 | 描述 |
|------|------|
| **身份** | 29 岁男性，德国慕尼黑量子计算实验室博士后，英语母语；Royal Road + Reddit（r/HFY, r/progressionfantasy）双栖用户；阅读伴随技术笔记（Notion 中同步更新“本章科技树缺口”）；已读《Rendezvous with Rama》《The Three-Body Problem》（英文版），对《Expanse》系列剧集有深度考据癖 |
| **阅读习惯** | 每日1–2章，但每章耗时较长（平均25分钟/章）；必查作者注释与附录科技词典；对“科学错误”容忍度极低（例：“核聚变电池仅拳头大却供能十年”→ 若未说明“利用真空零点能耦合”则视为崩坏）；反感“科技即魔法”（所有超常现象必须有可追溯的技术接口） |
| **喜欢什么** | 技术突破推动剧情（主角修复废弃传感器→发现地壳微震规律→预测火山喷发→拯救殖民地→获准接触禁忌数据库）；原理融入叙事（描写“神经织网”时不写“意识上传”，而写“突触电位差降至阈值以下，海马体γ波同步率骤升37%，他‘看见’了三小时前自己遗忘的密码”）；AI角色有可信限制（导航AI能规划最优路径，但无法理解“为何人类愿为一朵花绕行两公里”）；太空背景细节扎实（舱内气压维持逻辑、辐射屏蔽材料衰减周期、通讯延迟对战术的影响） |
| **讨厌什么** | 硬科幻沦为名词堆砌（大段“量子纠缠态生物计算机”术语无上下文）；技术设定前后矛盾（前文说“曲速引擎需冷却72小时”，后文3小时内连续跃迁）；主角突然掌握未铺垫科技（第45章凭空造出反物质发生器，此前0线索）；科学错误低级（“火星重力为地球38%”写成“3.8%”） |
| **耐心阈值** | **Hook 窗口：≤3章**——第1章须明确科技基调（如：“‘普罗米修斯号’主控AI死机，船员用万用表测出神经织网接口电压异常：+5.02V（标称+5.00V），误差0.4%——足够让AI拒绝执行所有指令”）；第2章须展示主角用基础原理破局（他拆解备用电源，手动调节电压至+5.001V，AI重启，但只承认“校准有效”，拒绝解释原因）；第3章须引出技术规则的深层代价（AI恢复后静默37秒，期间飞船自动规避了3次微陨石——而船员不知它何时开始监控他们）；**弃书触发点**：技术解决方案违背已立规则、关键科技突破无前置知识储备描写（主角从未接触过电路却修好量子阵列）、科学细节经不起基础推演（Reddit 评论区10分钟内指出漏洞） |
| **判断标准** | 能否用中学物理/化学知识，大致推演出本章技术方案的可行性边界？若答案为“可以，且作者留的余量合理”，则留存；若答案为“这需要推翻热力学第二定律”，则弃书。**终极检验是技术逻辑的可证伪性**。 |
